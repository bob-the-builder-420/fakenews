Fake news checker ----------------- A tool created by Ben Engdahl and Kaleb Wells from the National Inventors Hall of Fame Middle School. Installation ------------ Our extension is not currently on the Google Chrome Web Store, however, it can be temporarily installed through [our Github](https://github.com/kwells248/fakenews). 1. Look for the green *Clone or Download* Button. Click the button and then select *Download ZIP* 2. Open the *fakenews-master.zip* file in a file manager. 3. Right click the *fakenews-master.zip* file and select *Extract All*. In the dialog box that appears, select Extract. 4.
